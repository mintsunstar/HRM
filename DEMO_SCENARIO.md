# 데모 시나리오 가이드

## 전체 데모 시나리오 (약 10-15분)

### 1. 로그인 및 대시보드 (2분)

#### 1.1 Super Admin 로그인
1. 브라우저에서 `/login` 접속
2. 이메일: `superadmin@bdgen.co.kr`, 비밀번호: `pass1234` 입력
3. 로그인 버튼 클릭
4. 대시보드로 자동 이동 확인

#### 1.2 대시보드 확인
- 오늘 전사 근태 요약 카드 확인 (전체/출근/지각/결근/휴가)
- 출근율 차트 확인 (오늘/이번 주/이번 달)
- 부서별 출근율 확인
- 이상 근태 알림 리스트 확인
- 최근 활동 로그 5건 확인
- 카드 클릭 시 해당 페이지로 이동 확인

### 2. 직원 관리 (3분)

#### 2.1 직원 목록 조회
1. 사이드바에서 "직원 관리" 클릭
2. 검색 기능 테스트 (이름, 이메일, 사번)
3. 부서 필터 테스트
4. 상태 필터 테스트 (활성/비활성)
5. 페이지네이션 동작 확인

#### 2.2 직원 초대
1. "직원 초대" 버튼 클릭
2. 모달에서 정보 입력:
   - 이메일: `test@bdgen.co.kr`
   - 이름: `테스트 사용자`
   - 사번: `EMP999`
   - 부서: `개발팀`
   - 직책: `개발자`
   - 권한 레벨: `User (Level 3)`
3. "초대" 버튼 클릭
4. 성공 토스트 확인
5. 리스트에 새 직원 추가 확인

#### 2.3 중복 체크 테스트
1. 다시 "직원 초대" 클릭
2. 동일한 이메일 입력 → 중복 오류 확인
3. 동일한 사번 입력 → 중복 오류 확인

#### 2.4 직원 정보 수정
1. 리스트에서 직원 선택 → "수정" 클릭
2. 이름, 전화번호 등 수정
3. "저장" 클릭
4. 리스트에 반영 확인

#### 2.5 비밀번호 초기화
1. 직원 선택 → "비밀번호 초기화" 클릭
2. 확인 모달에서 "초기화" 클릭
3. 성공 토스트 확인
4. 활동 로그에 기록 확인

#### 2.6 Excel 다운로드
1. "Excel 다운로드" 버튼 클릭
2. 파일 다운로드 확인
3. Excel 파일 열어서 데이터 확인

### 3. 전사 근태 관리 (3분)

#### 3.1 근태 목록 조회
1. 사이드바에서 "전사 근태 관리" 클릭
2. 기간 선택 (이번 달)
3. "조회" 버튼 클릭
4. 검색 기능 테스트 (이름, 사번)
5. 상태 필터 테스트

#### 3.2 근태 수정 (Super Admin)
1. 최근 30일 이내 근태 데이터 선택 → "수정" 클릭
2. 수정 모달에서:
   - 출근시간 수정
   - 퇴근시간 수정
   - 상태 변경
   - 수정 사유 입력 (10~200자)
3. "저장" 클릭
4. 성공 토스트 확인
5. 테이블에 반영 확인
6. 활동 로그에 기록 확인

#### 3.3 수정 기간 제한 확인
1. 30일 이전 데이터 선택 → "수정 불가" 표시 확인
2. (Admin 계정으로 전환 시) 7일 이전 데이터 수정 불가 확인

#### 3.4 CAPS 업로드 (Super Admin만)
1. "CAPS 업로드" 버튼 클릭
2. 파일 선택 (CSV/Excel)
3. 적용 옵션 선택:
   - 신규 추가
   - 덮어쓰기
   - 전체 삭제 후 업로드
4. "업로드" 클릭
5. 성공 토스트 확인
6. 활동 로그에 기록 확인

#### 3.5 Excel 다운로드
1. "Excel 다운로드" 버튼 클릭
2. 파일 다운로드 확인

### 4. 통계 및 리포트 (2분)

#### 4.1 리포트 생성
1. 사이드바에서 "통계 및 리포트" 클릭
2. "리포트 생성" 버튼 클릭
3. 모달에서:
   - 리포트명 입력
   - 리포트 유형 선택 (일간/주간/월간/사용자 정의)
   - 기간 선택
4. "생성" 클릭
5. 성공 토스트 확인
6. 리포트 리스트에 추가 확인

#### 4.2 리포트 다운로드
1. 생성된 리포트의 "다운로드" 버튼 클릭
2. (준비 중 메시지 확인)

### 5. 활동 로그 (1분)

#### 5.1 로그 조회
1. 사이드바에서 "활동 로그" 클릭
2. 필터 테스트 (대상 유형별)
3. 페이지네이션 확인
4. 최근 활동 내역 확인:
   - 로그인
   - 직원 초대/수정
   - 근태 수정
   - 리포트 생성
   - 시스템 설정 변경

### 6. 시스템 설정 (Super Admin만) (2분)

#### 6.1 근무 정책 설정
1. 사이드바에서 "시스템 설정" 클릭 (Super Admin만 표시)
2. 근무 정책 섹션:
   - 출근 시간 수정
   - 퇴근 시간 수정
   - 점심 시간 수정
   - 지각 기준 수정

#### 6.2 블록체인 설정
1. 블록체인 연동 활성화/비활성화
2. 엔드포인트 입력

#### 6.3 CAPS 설정
1. CAPS 연동 활성화/비활성화
2. 동기화 간격 설정

#### 6.4 알림 설정
1. 이메일 알림 설정
2. 일일 요약 설정

#### 6.5 설정 저장
1. "저장" 버튼 클릭
2. 확인 모달에서 "저장" 클릭
3. 성공 토스트 확인
4. 활동 로그에 기록 확인

### 7. 권한 테스트 (2분)

#### 7.1 Admin 계정으로 전환
1. 로그아웃
2. Admin 계정으로 로그인 (`admin@bdgen.co.kr` / `pass1234`)

#### 7.2 권한 제한 확인
1. 사이드바에서 "시스템 설정" 메뉴 비표시 확인
2. "CAPS 업로드" 버튼 비표시 확인
3. 직원 관리에서 자신의 부서만 표시 확인
4. 근태 관리에서 7일 이전 데이터 수정 불가 확인
5. 활동 로그에서 본인 로그만 표시 확인

### 8. 내 정보 (1분)

#### 8.1 내 정보 수정
1. 헤더의 사용자 메뉴 클릭
2. "내 정보" 클릭
3. 이름, 이메일, 전화번호 수정
4. 부서는 읽기 전용 확인
5. "저장" 클릭
6. 성공 토스트 확인

### 9. 반응형 테스트 (1분)

#### 9.1 모바일 뷰
1. 브라우저 개발자 도구에서 모바일 뷰로 전환
2. 사이드바 토글 버튼 확인
3. 테이블 가로 스크롤 확인
4. 모달 반응형 확인
